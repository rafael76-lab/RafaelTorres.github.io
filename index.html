Predicción del valor de autos usados con Machine Learning

El proyecto consistió en desarrollar un modelo de machine learning para Rusty Bargain, una plataforma de venta de autos usados, con el objetivo de estimar el valor de mercado de los vehículos de forma rápida y precisa dentro de su aplicación; trabajé con un dataset que incluía información técnica y de uso de los autos, realizando un proceso de limpieza de datos que incluyó la eliminación de valores nulos, descarte de variables irrelevantes y la creación de una nueva variable geográfica a partir del código postal, seguido de la construcción y comparación de múltiples modelos como regresión lineal (baseline), Random Forest y algoritmos de gradient boosting como LightGBM y CatBoost, aplicando diferentes técnicas de codificación de variables categóricas (one-hot encoding y label encoding), además de medir no solo la calidad predictiva mediante RMSE sino también los tiempos de entrenamiento para evaluar su viabilidad en producción; los resultados mostraron que LightGBM logró el mejor desempeño global al ofrecer el menor error con tiempos de entrenamiento eficientes, superando a otros modelos y permitiendo recomendarlo como la mejor opción para implementación en la aplicación.

Tecnologías Python; pandas; NumPy; Scikit-learn; LightGBM; CatBoost; GitHub; Git

Estimación de edad con visión por computadora para control de venta de alcohol

El proyecto consistió en evaluar la viabilidad de utilizar modelos de visión por computadora para ayudar a la cadena de supermercados Good Seed a cumplir con la regulación que prohíbe la venta de alcohol a menores de edad, mediante la estimación automática de la edad de los clientes a partir de imágenes capturadas en el punto de venta; trabajé con un dataset de fotografías etiquetadas con la edad real, realizando un análisis exploratorio inicial para comprender la distribución de edades y la calidad de las imágenes, seguido de la construcción de un pipeline de entrenamiento en GPU utilizando TensorFlow y Keras, donde implementé técnicas de aumento de datos (rotaciones, zoom, desplazamientos) para mejorar la generalización del modelo, y desarrollé un modelo basado en transfer learning utilizando EfficientNetB3 preentrenado en ImageNet, al que se le añadió una cabeza personalizada con capas densas y regularización mediante dropout; el entrenamiento se realizó en dos fases, primero congelando la red base para entrenar únicamente la cabeza del modelo y posteriormente aplicando fine-tuning desbloqueando parcialmente las capas profundas para mejorar la precisión, utilizando función de pérdida Huber y la métrica MAE, junto con early stopping para evitar sobreajuste; los resultados demostraron que el modelo es capaz de estimar la edad con un error razonable, validando la hipótesis de que este enfoque puede ser utilizado como sistema de apoyo para detectar posibles menores de edad en tiempo real, proporcionando una base sólida para su implementación en un entorno productivo.

Tecnologías Python; pandas; TensorFlow; Keras; EfficientNet; Computer Vision; Data Augmentation; GPU; GitHub; Git

Clasificación de planes móviles con Machine Learning

El proyecto consistió en desarrollar un modelo de clasificación para la empresa de telecomunicaciones Megaline, con el objetivo de recomendar automáticamente el plan más adecuado (Smart o Ultra) a sus clientes en función de su comportamiento mensual, ayudando así a migrar usuarios desde planes heredados hacia nuevas ofertas; trabajé con un dataset que incluía variables como número de llamadas, minutos consumidos, mensajes y uso de datos móviles, realizando la segmentación del conjunto en entrenamiento, validación y prueba para garantizar una evaluación robusta del modelo, seguido de la implementación y comparación de distintos algoritmos de clasificación como árbol de decisión, regresión logística y Random Forest, ajustando hiperparámetros clave como la profundidad del árbol y el número de estimadores para optimizar el desempeño, además de construir un modelo baseline basado en la moda para validar que los modelos entrenados capturaran patrones reales; la evaluación se realizó utilizando la métrica de exactitud, donde el modelo Random Forest obtuvo el mejor desempeño alcanzando un accuracy de 0.79 en el conjunto de prueba, superando el umbral requerido de 0.75, mientras que el árbol de decisión mostró resultados competitivos y la regresión logística tuvo un rendimiento inferior debido a su naturaleza lineal, confirmando que los modelos basados en árboles son más adecuados para este tipo de problema; como resultado, se identificó un modelo capaz de mejorar la toma de decisiones comerciales y optimizar la asignación de planes a clientes en un entorno real.

Tecnologías Python; pandas; Scikit-learn; Machine Learning; Clasificación; GitHub; Git

Análisis de datos de viajes compartidos y prueba de hipótesis con SQL y Python

El proyecto consistió en realizar un análisis de datos para Zuber, una empresa de viajes compartidos en Chicago, con el objetivo de identificar patrones en el comportamiento de los usuarios, analizar la actividad de empresas competidoras y evaluar el impacto de factores externos como el clima en los viajes; trabajé con una base de datos relacional que incluía información de viajes, taxis, barrios y registros meteorológicos, utilizando SQL para realizar consultas complejas que permitieron calcular el número de viajes por empresa en periodos específicos, agrupar y segmentar compañías relevantes, y vincular datos de clima con viajes mediante joins por timestamp, además de clasificar condiciones meteorológicas en categorías como “Good” y “Bad”; posteriormente, complementé el análisis en Python con pandas para explorar datasets adicionales, identificar los barrios con mayor número de finalizaciones y generar visualizaciones que facilitaran la interpretación de los datos; finalmente, se llevó a cabo una prueba de hipótesis para evaluar si la duración de los viajes entre el Loop y el aeropuerto O’Hare cambia durante los sábados lluviosos, definiendo hipótesis nula y alternativa, seleccionando un nivel de significancia y aplicando pruebas estadísticas, lo que permitió confirmar diferencias en la duración bajo condiciones climáticas adversas y generar insights relevantes para la toma de decisiones estratégicas.

Tecnologías SQL; Python; pandas; SciPy; Plotly; Análisis de datos; Estadística
