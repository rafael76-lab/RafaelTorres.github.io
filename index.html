Predicción del valor de autos usados con Machine Learning

Desafío

Desarrollar un modelo para estimar el valor de mercado de autos usados en Rusty Bargain.

Proceso

Limpieza de datos, manejo de nulos, feature engineering desde código postal, entrenamiento de modelos (Linear Regression, Random Forest, LightGBM, CatBoost), y evaluación con RMSE.

Resultado y visuales

LightGBM obtuvo el menor RMSE y mejor eficiencia computacional.

Lección o hallazgos

Boosting supera modelos clásicos en problemas de regresión tabular.

Python · pandas · Scikit-learn · LightGBM · CatBoost

Desafío

Proceso

Dataset de imágenes, data augmentation, transfer learning con EfficientNetB3, entrenamiento en dos fases y regularización con dropout.

Resultado y visuales

Modelo con MAE competitivo, viable para uso como sistema de apoyo.

Lección o hallazgos

Transfer learning reduce drásticamente costo de entrenamiento en visión por computadora.

TensorFlow · Keras · EfficientNet · Computer Vision
